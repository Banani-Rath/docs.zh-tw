# My.Computer.Keyboard Object
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

提供用於存取鍵盤目前狀態 \(例如，目前按下的是哪個鍵\) 的屬性，並提供將按鍵傳送至使用中視窗的方法。  
  
## 備註  
 如需有關的方法和屬性的資訊`My.Computer.Keyword`物件，請參閱<xref:Microsoft.VisualBasic.Devices.Keyboard>。  
  
 如需詳細資訊，請參閱 [Accessing the Keyboard](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-keyboard.md)。  
  
## 需求  
 **命名空間︰** <xref:Microsoft.VisualBasic.Devices>  
  
 **類別︰** <xref:Microsoft.VisualBasic.Devices.Keyboard>  
  
 **組件**：Visual Basic 執行階段程式庫 \(在 Microsoft.VisualBasic.dll 中\)
