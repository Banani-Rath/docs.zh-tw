# Overflow (Visual Basic Error)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

常值代表其所指定資料型別限制以外的數值。  
  
 **錯誤 ID：**BC30036  
  
### 若要更正這個錯誤  
  
-   請查閱目標資料型別的數值範圍並重寫該常值，以便符合範圍。
