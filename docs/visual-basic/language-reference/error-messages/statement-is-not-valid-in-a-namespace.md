# Statement is not valid in a namespace
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

陳述式不能顯示在命名空間的層級。  命名空間唯一允許的宣告是模組、介面、類別、委派、列舉型別和結構宣告。  
  
 **錯誤 ID︰**BC30001  
  
### 若要更正這個錯誤  
  
-   將陳述式移動到模組、類別、介面、結構、列舉型別或委派定義中的位置。
