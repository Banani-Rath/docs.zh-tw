# Some subkeys cannot be deleted
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

已嘗試刪除登錄機碼，但因無法刪除某些子機碼而導致作業失敗。  這通常是因為缺少使用權限所造成。  
  
### 若要更正這個錯誤  
  
-   確定您有足夠的使用權限能夠刪除指定的子機碼。
