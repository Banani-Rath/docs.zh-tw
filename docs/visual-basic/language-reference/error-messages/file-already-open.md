# File already open
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

有時候檔案必須先關閉，才能進行另一個 `FileOpen` 或其他作業。  可能導致本錯誤的各種原因包括：  
  
-   為已開啟的檔案，執行循序輸出模式 `FileOpen` 作業  
  
-   陳述式參考至已開啟的檔案。  
  
### 若要更正這個錯誤  
  
1.  請在執行陳述式之前先關閉該檔案。
