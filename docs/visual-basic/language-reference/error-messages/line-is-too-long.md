# Line is too long
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

原始程式文字不能超過 65535 個字元。  
  
 **錯誤 ID︰**BC30494  
  
### 若要更正這個錯誤  
  
-   將行長度縮短為 65535 個字元或更少。
