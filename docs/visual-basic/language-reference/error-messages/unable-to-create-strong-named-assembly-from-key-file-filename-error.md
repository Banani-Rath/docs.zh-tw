# Unable to create strong-named assembly from key file &#39;&lt;filename&gt;&#39;: &lt;error&gt;
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

無法從指定金鑰檔中建立強式名稱組件。  
  
 **錯誤 ID︰**BC31026  
  
### 若要更正這個錯誤  
  
1.  確認已經指定正確的金鑰檔，且沒有被另一個應用程式鎖住。
