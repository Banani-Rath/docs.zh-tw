# XML entity references are not supported
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

未在 XML 1.0 規格中定義的實體參考 \(例如，`©`\) 都會當做 XML 常值的值而加入。  XML 常值中僅支援 `&`、`"`、`<`、`>` 和 `'` XML 實體參考。  
  
 **錯誤 ID**：BC31180  
  
### 若要更正這個錯誤  
  
-   移除不支援的實體參考。
