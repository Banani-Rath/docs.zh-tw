# Type mismatch (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

您嘗試將數值轉換為其他型別，然而轉換無效。  
  
### 若要更正這個錯誤  
  
1.  請檢查設定以確定是否有效。  
  
2.  請確定您並未將物件傳遞至需要單一屬性 \(Property\) 或值的程序。  
  
3.  請確定您並未在應該是運算式的位置上使用模組或專案名稱。
