# &#39;Optional&#39; expected
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

程序宣告中的選擇性引數之後加上一個必要的引數。  選擇性引數之後的每個引數必須是選擇性的。  
  
 **錯誤 ID：**BC30202  
  
### 若要更正這個錯誤  
  
1.  如果引數變成是必須的，請將它移動到引數清單的第一個選擇性引數之前。  
  
2.  如果引數是選擇性的，請使用 `Optional` 關鍵字。
