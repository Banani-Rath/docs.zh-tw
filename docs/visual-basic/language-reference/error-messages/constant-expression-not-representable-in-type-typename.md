# Constant expression not representable in type &#39;&lt;typename&gt;&#39;
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

您嘗試評估無法符合目標型別的常數，通常是因為常數溢位而超出範圍。  
  
 **錯誤 ID**：BC30439  
  
### 若要更正這個錯誤  
  
1.  將目標型別變更為可處理常數的型別。
