# /help, /? (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

顯示編譯器選項。  
  
## 語法  
  
```  
/help  
' -or-  
/?  
```  
  
## 備註  
 如果您在編譯中包含這個選項，不會建立輸出檔，也不會進行編譯。  
  
> [!NOTE]
>  `/help` 選項無法在 [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] 開發環境之中使用，只適用於從命令列編譯時。  
  
## 範例  
 下列程式碼會從命令列顯示說明。  
  
```  
vbc /help  
```
