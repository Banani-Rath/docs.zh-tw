# /nologo (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

隱藏在編譯期間顯示的著作權橫幅和資訊訊息。  
  
## 語法  
  
```  
/nologo  
```  
  
## 備註  
 如果您指定 `/nologo`，編譯器 \(Compiler\) 不會顯示著作權橫幅。  `/nologo` 預設為不作用。  
  
> [!NOTE]
>  `/nologo` 選項無法在 Visual Studio 開發環境內使用，只有在命令列編譯時才能使用。  
  
## 範例  
 下列程式碼會編譯 `T2.vb`，而且不會顯示著作權橫幅。  
  
```  
vbc /nologo t2.vb  
```
