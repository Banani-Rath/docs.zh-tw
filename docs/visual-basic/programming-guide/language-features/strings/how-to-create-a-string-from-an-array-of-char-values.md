## 編譯程式碼  
 這個方法沒有任何特殊的需求。  
  
 語法 `"a"c` \(其中單一 `c` 緊跟著以引號括住的單一字元\) 是用來建立字元常值 \(Character Literal\)。  
  
## 穩固程式設計  
 字串中的 null 字元 \(相當於 `Chr(0)`\) 會在使用字串時造成無法預期的結果。  null 字元包含在字串中，但是在某些情況下不會顯示接在 null 字元後的字元。
