# 項目名稱不能使用 &#39;xmlns&#39; 前置字元
使用 XML 命名空間前置字元 `xmlns` 指定了 XML 元素常值。 例如:  
  
```vb#  
Dim elem = <xmlns:ElementName>  
```  
  
 XML 1.0 規格會將 `xmlns` 識別為保留字。  
  
 **錯誤 ID︰**BC31189  
  
### 更正這個錯誤  
  
-   將 XML 命名空間前置字元變更為有效值，或移除前置字元。
