# 無法讀取分隔的欄位，因為分隔符號為 Nothing 或空白
`TextFieldParser` 無法讀取自檔案，因為 `Delimiters` 屬性設定為 `Nothing` 或空白。  
  
### 更正這個錯誤  
  
-   請提供 `Delimiters` 的有效值。
