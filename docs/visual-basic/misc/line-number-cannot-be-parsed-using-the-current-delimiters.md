# 無法使用目前的分隔符號剖析第 &lt;數字&gt; 行
無法剖析指定的行，因為使用了未指定的分隔符號。  
  
### 更正這個錯誤  
  
-   調整 `Delimiters` 以便正確地剖析此行，或插入例外狀況處理程式碼以便處理此行。
