# 匿名類型至少必須包含一個成員
匿名類型宣告中的初始設定式清單不可為空白。  
  
```  
' Not valid. ' Dim anonInstance = New With {}  
```  
  
 **錯誤 ID︰**BC36574  
  
### 更正這個錯誤  
  
-   在大括號內宣告一個成員，如下列程式碼所示。  
  
    ```  
    Dim anonInstance = New With {.MemberName = "value"}  
    ```
