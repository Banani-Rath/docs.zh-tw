# XML 實體必須以 &#39;;&#39; 結尾
XML 實體參考 \(例如 `>`\) 遺漏結尾分號 \(;\)。  
  
 **錯誤 ID︰**BC31178  
  
### 更正這個錯誤  
  
-   在 XML 實體參考結尾加入分號。
