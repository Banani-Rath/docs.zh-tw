# 必須有對應的 XML 屬性值右雙引號
以雙引號分隔的屬性值遺漏右引號字元。  
  
 **錯誤 ID：**BC31164  
  
### 更正這個錯誤  
  
-   加入右雙引號字元來分隔屬性值。
