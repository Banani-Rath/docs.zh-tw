# &#39;&lt;事件名稱&gt;&#39; 不是 &#39;&lt;容器名稱&gt;&#39; 的事件
未針對物件宣告指定的事件。  
  
 **錯誤 ID：**BC30676  
  
### 更正這個錯誤  
  
1.  檢查事件名稱的拼字。  
  
2.  確認您存取的物件正確。 使用完整的物件參考，或視需要使用 `Imports` 陳述式匯入適當的命名空間。
