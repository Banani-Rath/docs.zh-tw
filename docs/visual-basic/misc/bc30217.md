# 必須有字串常數。
雙引號 \("\) 以外的字元發生在內容需要字串常數時。 其中一個可能的原因是字串遺漏引號。  
  
 **錯誤 ID︰**BC30217  
  
### 更正這個錯誤  
  
1.  請確定正確地以雙引號括住字串。  
  
2.  提供字串常數。
