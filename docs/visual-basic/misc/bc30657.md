# &#39;&lt;程序名稱&gt;&#39; 含有一個不支援的傳回類型或多個不支援的參數類型
宣告的程序具有 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] 無法代表的參數或傳回類型。  
  
 **錯誤 ID︰**BC30657  
  
### 更正這個錯誤  
  
-   將傳回類型或參數定義為支援的類型。  
  
-   開發可描述這個程序所使用資料的新類型。
