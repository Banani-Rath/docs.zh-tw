# 必須是 &#39;}&#39;
陣列初始設定式或條件約束清單未以有效的形式結束。  
  
 用來初始化陣列的項目值必須括在大括弧 \(`{}`\) 內。  
  
```  
Dim demoArray() As Integer = New Integer() {1, 2, 3}   
```  
  
 同樣地，泛型類型參數之條件約束清單中的條件約束必須括在大括弧內。  
  
```  
Public Class dictionaryMaker(Of t As {IComparable, IDisposable, New})   
```  
  
 **錯誤 ID︰**BC30370  
  
### 更正這個錯誤  
  
-   使用 "}" 結束陣列初始設定式或條件約束清單。
