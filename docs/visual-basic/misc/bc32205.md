# WithEvents 變數名稱長度不能超過 1019 個字元
使用 `WithEvents` 宣告的每個變數名稱或識別項限制為 1019 個字元。  
  
 **錯誤 ID：**BC32205  
  
### 更正這個錯誤  
  
-   減少識別項的長度。
