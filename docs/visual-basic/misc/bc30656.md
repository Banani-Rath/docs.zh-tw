# 欄位 &#39;&lt;欄位名稱&gt;&#39; 是屬於不支援的類型
欄位宣告為無法透過 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] 代表的類型。  
  
 **錯誤 ID︰**BC30656  
  
### 更正這個錯誤  
  
1.  將欄位定義為支援的類型。  
  
2.  開發可描述欄位中資料的新類型。
