# 分隔符號不可以是 Nothing 或空字串
`TextFieldParser` 無法讀取自檔案，因為 `Delimiters` 屬性設定為 `Nothing` 或為空白 `String` \(""\)。  
  
### 更正這個錯誤  
  
-   請提供 `Delimiters` 的有效值。
