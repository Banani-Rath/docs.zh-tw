# 檔案 &#39;&lt;檔案名稱&gt;&#39; 不是文字檔
無法寫入指定的檔案，因為它是二進位檔。  
  
 **錯誤 ID︰**BC2015  
  
### 更正這個錯誤  
  
-   請檢查您提供的檔名和路徑是否正確。
