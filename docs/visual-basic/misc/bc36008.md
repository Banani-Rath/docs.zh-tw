# &#39;Using&#39; 之後必須搭配相對應的 &#39;End Using&#39;
`Using` 陳述式沒有相對應的 `End Using` 陳述式。  
  
 必須使用 `End Using` 陳述式來結束 `Using` 區塊。  
  
 **錯誤 ID︰**BC36008  
  
### 更正這個錯誤  
  
1.  如果這個 `Using` 區塊是一組巢狀 `Using` 區塊的一部分，請確定已正確地終止每個區塊。  
  
2.  將 `End Using` 陳述式加入 `Using` 區塊的結尾。
