# 泛型方法內的區域變數不可以宣告為 &#39;Static&#39;
泛型程序內的區域變數宣告會指定 `Static`。  
  
 Visual Basic 和 .NET Framework 目前不支援任何的靜態變數和泛型程序組合。  
  
 **錯誤 ID︰**BC32068  
  
### 更正這個錯誤  
  
-   請從變數宣告中移除 `Static` 關鍵字。
