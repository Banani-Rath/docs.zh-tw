# 無法在具有 &#39;As New&#39; 的變數宣告中指定可為 Null 的修飾詞
在指定 `As New` 的變數宣告中已包括可為 Null 的類型修飾詞 \(?\)。 下列範例會產生這個錯誤：  
  
```vb#  
Dim num? As New ExampleStructure  
```  
  
 **錯誤 ID︰**BC33109  
  
### 更正這個錯誤  
  
1.  請從可為 Null 的變數宣告中移除 `New` 關鍵字 \(如下列範例所示\)：  
  
    ```vb#  
    Dim num? As ExampleStructure  
    ```
