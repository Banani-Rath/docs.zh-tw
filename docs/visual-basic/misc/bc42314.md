# 在部分 &lt;類型&gt; 上不可套用多次 XML 註解
在部分 \<類型\> 上不可套用多次 XML 註解。 將會忽略這個 \<類型\> 的 XML 註解。  
  
 XML 註解區塊只能出現在部分類型的某個部分上方。  
  
 如果 XML 註解區塊出現在部分類型的多個部分上方，則會針對每個註解區塊建立這個警告，並忽略最上層 XML 註解。  
  
 **錯誤 ID︰**BC42314  
  
### 更正這個錯誤  
  
-   請移除多餘的註解區塊。
