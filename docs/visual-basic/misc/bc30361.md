# &#39;&lt;屬性1&gt;&#39; 和 &#39;&lt;屬性2&gt;&#39; 無法互相多載，因為只有一個宣告為 &#39;Default&#39;
如果屬性指定 `Default`，該名稱上多載的所有屬性也必須指定 `Default`。  
  
 **錯誤 ID︰**BC30361  
  
### 更正這個錯誤  
  
-   確定將所有多載的屬性宣告為 `Default`。
