# 實作類型必須是一個介面
`Implements` 陳述式不會指定介面。 一個類別僅可以實作一個介面。  
  
 **錯誤 ID：**BC30232  
  
### 更正這個錯誤  
  
1.  請確定介面名稱拼寫正確。  
  
2.  如果陳述式會指定類別，請考慮使用 `Inherits` 陳述式。
