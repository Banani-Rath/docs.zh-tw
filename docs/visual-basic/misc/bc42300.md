# XML 註解區塊必須緊接在所套用的語言項目之前
XML 註解區塊必須緊接在所套用的語言元素之前。 將會忽略 XML 註解。  
  
 如果標記錯置或格式不正確，也會發生這個錯誤。  
  
 **錯誤 ID︰**BC42300  
  
### 更正這個錯誤  
  
1.  將註解區塊移至所套用的語言項目之前。 確定未在初始標記之前不小心插入多餘的字元。  
  
2.  確定標記有效。
