# &#39;Implements&#39; 在模組中無效
模組內發生 `Implements` 陳述式或子句。 模組不能實作介面。  
  
 **錯誤 ID︰**BC30231  
  
### 更正這個錯誤  
  
-   請移除 `Implements` 陳述式或子句，或將模組重新輸入為類別。
