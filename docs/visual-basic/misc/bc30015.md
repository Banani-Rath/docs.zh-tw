# 從 &#39;System.&lt;類別名稱&gt;&#39; 繼承無效
類別不可從 `System.Array`、`System.Delegate`、`System.Enum` 或 `System.ValueType` 繼承。  
  
 **錯誤 ID︰**BC30015  
  
### 更正這個錯誤  
  
-   移除 `Inherits` 陳述式，或將其變更為繼承自有效的基底類別。
