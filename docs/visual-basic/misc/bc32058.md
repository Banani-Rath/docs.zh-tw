# 類型引數無法套用至運算式 &#39;&lt;運算式&gt;&#39;
使用將類型引數傳遞給匯入別名的 [Of](../../visual-basic/language-reference/statements/of-clause.md) 子句，定義匯入別名。  
  
 類型引數用於泛型類型，而且只有類別、結構、介面、程序和委派才能是泛型。 命名空間和匯入別名可以是泛型。  
  
 **錯誤 ID：**BC32058  
  
### 更正這個錯誤  
  
-   請從匯入別名中移除 `Of` 子句和其類型引數。
