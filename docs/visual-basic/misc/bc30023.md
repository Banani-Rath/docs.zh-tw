# 宣告為 &#39;WriteOnly&#39; 的屬性無法擁有 &#39;Get&#39;
`Get` 程序會讀取屬性值。 無法讀取 `WriteOnly` 屬性。  
  
 **錯誤 ID：**BC30023  
  
### 更正這個錯誤  
  
-   請從 `Property` 陳述式中移除 `WriteOnly` 關鍵字，或從屬性主體中移除 `Get` 程序。
