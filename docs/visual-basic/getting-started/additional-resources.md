### 網站  
  
|詞彙|定義|  
|--------|--------|  
|[Microsoft Visual Basic 開發人員中心](http://go.microsoft.com/fwlink/?LinkID=47768)|提供程式碼範例、升級資訊、影片、免費下載和技術性內容。|

|[Microsoft 模式和做法](http://msdn.microsoft.com/practices/default.aspx)|提供應用工程指引，包括生產環境品質的原始程式碼和文件。|
