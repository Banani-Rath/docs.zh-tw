 請參閱個別主題的詳細資訊和範例。  
  
 雖然編譯器不需要個別的前置處理器，但這一節所述的指示詞會處理，好像發生了一個。  它們用來協助在條件式編譯。  與 C 和 C\+\+ 的指示詞不同，您不能用這些指示詞建立巨集。  
  
 前置處理器指示詞必須是每一行中唯一的指令。
