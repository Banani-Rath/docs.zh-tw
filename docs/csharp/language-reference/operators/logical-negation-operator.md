# ! 運算子 (C# 參考)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

邏輯負運算子 \(`!`\) 是將運算元做否定運算的一元運算子。  它是為了 `bool` 而定義的，只有在其運算元為 `false` 時才會傳回 `true`。  
  
## 備註  
 使用者定義型別可多載 `!` 運算子 \(請參閱 [operator](../../../csharp/language-reference/keywords/operator.md)\)。
