# -&gt; 運算子 (C# 參考)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`->` 運算子結合了指標取值和成員存取。  
  
## 備註  
 這種形式的運算式，  
  
```  
x->y  
```  
  
 \(其中 `x` 為 `T*` 型別的指標，而 `y` 為 `T` 的成員\) 相等於  
  
```  
(*x).y  
```  
  
 `->` 運算子只能在標示為 [unsafe](../../../csharp/language-reference/keywords/unsafe.md) 的程式碼中使用。  
  
 `->` 運算子無法多載。
