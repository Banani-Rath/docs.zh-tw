## 編譯程式碼

## 穩固程式設計  
 下列情形可能會造成例外狀況 \(Exception\)：  
  
-   檔案不在指定的位置上沒有或不存在。  檢查路徑和檔案名稱的拼字。  
  
## .NET Framework 安全性  
 不要依靠檔案名稱來判斷檔案內容。  例如，檔案 `myFile.cs` 可能不是 C\# 原始程式檔。
